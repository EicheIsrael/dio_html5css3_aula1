    Tags apresentadas nesse curso:

    !DOCTYPE html #Especifica o tipo do documento

    head.....#Informações gerais da pagina

    meta charset="utf-8" #Corrige problemas de acentos no portugues (não necessita fechar)

    title....#Para definir o nome da pagina atual na aba do navegador

    body.....#Area Principal do codigo

    nav......#Inicia um menu de navegação

    ul.......#Inicia uma lista

    il.......#Nomeia os itens da lista

    header...#Organiza o cabeçalho do site

    h1,2,3...#Exibe texto com titulo de paragrafo em tamanhos de 1 a 3...

    img......#Insere um imagem na pagina conforme o caminho em src

    .........atributo de img: width='???px' para redimencionar imagens em pixels proporcionais

    div......#Para apresentar imagens em linhas diferentes

    section..#Organiza uma sessão para melhor organização do codigo, id= para nomear

    a........#Inserir uma referencia dentro da própria página Ex: Ir ao topo

    .........atributo de a: href="NOME DA ID SECTION"

    .........atributo de a: href="mailto:">seu_endereço_de@email.aqui

    .........atributo de a: href="tel:SOMENTE_NUMERO_DO_TELEFONE"

    .........atributo de a: ..., target="_blank" para abrir o link em outra aba

    .........atributo de a: class="nome" para identificar uma class de atributos em css

    footer #Exibe um rodapé
